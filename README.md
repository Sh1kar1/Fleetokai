## Supported IDEs

- **IntelliJ IDEA**
- **PyCharm**
- **CLion**

## Installation

1. Open your IDE
2. Go to _Settings_ -> _Plugins_ -> _Marketplace_
3. Search for "**Fleetokai**"
4. Click _Install_
